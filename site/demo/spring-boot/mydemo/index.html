Simple spring boot kubernetes app

Steps:

  1. Create and run the spring boot app locally
  2. Package the application
  3. run maven install. Check that .jar file is under target/myemo-0.0.1-SNAPSHOT.jar.
  4. Create a DockerFile and create an image from it
  5. EXPOSE the port number
  6. Test the docker image
  7. Push the docker image to the docker hub "container repository".
